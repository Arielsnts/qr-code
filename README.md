# Qr Code Scanner

Aplicativo mobile desenvolvido com React Native e Expo como projeto de aprendizado. O app permite gerar e escanear QR Codes de forma simples e intuitiva.

Na tela de geração, o usuário pode inserir um texto, criar um QR Code correspondente e salvá-lo como imagem na galeria do dispositivo. Já na tela de scanner, é possível utilizar a câmera para ler QR Codes, visualizar o conteúdo decodificado e copiá-lo para a área de transferência.
